# Changelog

## v2.1.0 (2026-05-30)

### Added
- Emoji picker panel with 50+ emojis, accessible via the 😊 button.
- Click-outside-to-close behavior for the emoji panel.

### Changed
- Full Chinese localization of the user interface (placeholder texts, UI labels).
- Timestamps are now formatted in Chinese locale via `Intl.DateTimeFormat`.
- All code comments translated to Chinese.
- Default nickname changed to "匿名用户".

### Technical
- Single-file architecture preserved — no external dependencies.
- Emoji panel built with pure CSS and vanilla JavaScript.
- Original author attribution maintained.
